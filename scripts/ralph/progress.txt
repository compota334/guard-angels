- Cursor module: `src/messaging/cursors.ts` — `getCursor(projectRoot, angelId)`, `setCursor(projectRoot, angelId, offset)`. Single-line byte-offset files at `_cursors/<angel-id>`. Missing file → offset 0. Atomic write via tmpfile+rename.
- Cable module: `src/messaging/cables.ts` — `writeCable(projectRoot, data)`, `readInbox(projectRoot, angelId)`, `parseCable(filePath)`, `parseCableContent(raw)`. Writes to `_outbox/<from>/` first (audit trail), then copies to `_inbox/<to>/`.
